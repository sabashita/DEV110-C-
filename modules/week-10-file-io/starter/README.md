# Week 10 Starter — Habit Tracker (File I/O)

This is the starter folder for the Week 10 assignment.

## Quick Start

```bash
# Navigate here first:
cd modules/week-10-file-io/starter

# Run the program:
dotnet run

# Run tests:
dotnet test ../tests
```

> **Important:** Run `dotnet run` from inside this `starter/` folder so that `habits.csv` resolves to the correct path.

## Files

| File             | Your job                                |
| ---------------- | --------------------------------------- |
| `Program.cs`     | Implement the 6 TODO methods            |
| `Habit.cs`       | Fully provided — no changes needed      |
| `habits.csv`     | Sample data — feel free to edit it      |
| `STUDY_NOTES.md` | Fill in after completing the assignment |
