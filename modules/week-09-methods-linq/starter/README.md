# Week 9: Score Stats (Methods + LINQ) - Starter

## Summary

This is the **starter** project for Week 9.

You will build a small score-analysis app that practices **methods** (breaking a program into small helpers) and **LINQ** (Min/Max/Average/Count).

This version also includes a few short "report" lines so you can practice LINQ methods like `Where`, `OrderBy`, and `Take`.

## What You Need To Do

- Open `ScoreReport.cs`
- Follow the numbered TODO steps in the `ScoreReport` class
- Keep your prompts and messages **exactly** as required in the assignment README

---

## How To Run

```bash
cd modules/week-09-methods-linq/starter
dotnet run
```
